# Ecobazar - Organic eCommerce Shop

Ecobazar is a fully responsive e-commerce website for online food shopping, designed from Figma UI/UX prototypes. The website focuses on a clean layout, intuitive navigation, and smooth performance across desktop, tablet, and mobile devices.

## Features
- Home: Clear introduction and value proposition
- Products: Browse organic foods with categories
- Preview: Visual product showcase
- Cart & Checkout: User-friendly shopping experience
- About Us & Contact: Brand story and customer support

## Tech Stack
- HTML5 & CSS3
- JavaScript (vanilla JS)
- Figma for UI/UX design
